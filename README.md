# Padrões Atemporais — Guia do Engenheiro de Software

Material de estudo sobre os fundamentos de engenharia de software que não mudam com o tempo — independente de linguagem, framework ou stack.

## Conteúdo

1. **SQL** — A língua universal dos dados
2. **HTTP** — Como a web funciona desde 1991
3. **Estruturas de Dados** — Os blocos de construção
4. **Algoritmos** — Pensar em custo
5. **Git** — O modelo mental por baixo dos comandos
6. **Debugging & Logs** — O problema sempre está ali
7. **Padrões de Design** — Soluções com nome
8. **Concorrência** — O mesmo problema em toda linguagem
9. **Segurança** — Os ataques que não mudam
10. **Roadmap** — O que estudar e em que ordem

## Como usar

Acesse o guia completo em: **[padroes-atemporais.vercel.app](https://padroes-atemporais.vercel.app)**

Ou clone o repositório e abra o `index.html` direto no navegador — não há dependências de build.

```bash
git clone https://github.com/siquieroli/padroes-atemporais.git
cd padroes-atemporais
open index.html
```
